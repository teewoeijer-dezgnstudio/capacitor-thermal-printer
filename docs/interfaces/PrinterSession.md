# Interface: PrinterSession

Fluent builder returned by [`CapacitorThermalPrinterPlugin.useConnection`](CapacitorThermalPrinterPlugin.md#useconnection) and also exposed directly on the plugin instance. Each method queues ESC/POS commands for the targeted connection and returns the same session instance to facilitate chaining.

See [`CapacitorThermalPrinterPlugin`](CapacitorThermalPrinterPlugin.md) for the full list of formatting and content APIs that a session exposes.
